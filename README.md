# 🚗 Car Animation (HTML, CSS, JavaScript)

A creative car animation project using HTML, CSS, and JavaScript.  
It simulates a moving car with rotating wheels and a scrolling road background.

## 🚀 Features
- Moving road animation
- Rotating wheels
- Smooth CSS animations
- Interactive sound (optional with JavaScript)

## 🛠️ Tech Stack
- HTML
- CSS (Animations & Keyframes)
- JavaScript

## ▶️ How to Run
1. Clone or download the project
2. Open `index.html` in your browser

## 🎨 Animation Effects
- Road moves using CSS keyframes
- Wheels rotate continuously
- Background gives a motion effect

## 💡 Future Improvements
- Add speed control

- Add user interaction (start/stop button)
